Beacon control management system

ReactJS, Javascript, CSS, HTML

This is my graduation project which I developed during my internship at Tapme media Groningen. The most important part of this project is a mapeditor, which I developed completely on my own without using a framework. With this mapeditor customers are able to manage and place their beacons without interference from Tapme. Now that Tapme doesn't have to help customers with managing their beacons anymore they can focus on their other applications.

Safetyregion Groningen

Ionic, Javascript, CSS, HTML, Material design

With a group of fellow students we developed an application for the emergency services (firemen) in Groningen. In this application the leader of the firemens can give assignments to the firemens. For example they are able to give a drive-assignment to a nearby fire (with GPS location). We had two different parts in this application the control panel which was a website and an application which all firemens use. Whenever there is an emergency situation someone from the control panel can send a message to all "clients" firemen to go to a certain coördinate.

Guard Smartwatch

Tizen, Javascript, HTML, CSS, Java

Guard is a safety application that helps employees that work on their own. If they are in an emergency situation they are able to press a button which alarms the emergency services. I developed an application for the smartwatch, now bailiff's (Deurwaarders) are able to quickly press the button in case someone attacks them

Serious Game: Cleanawe

Flash, Adobe illustrator

Cleanawé is a platform game build for children in Africa. The goal of the game is to learn how to wash your hands. Since most children in Africa don't know how to wash their hands properly we were asked to create a game that teaches children how to wash their hands. It was a really hard game to develop since most children in Africa don't have any experience in playing games. The game has been released in Ghana Africa and children are able to play the game now.
